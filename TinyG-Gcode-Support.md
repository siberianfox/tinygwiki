# Gcode Cheat Sheet
This table summarizes Gcode supported

	Gcode | Command | Description
	---------|--------------|-------------
	G0 | Straight traverse | Traverse at maximum velocity to 10,20 
	G1 | Straight feed | 
	G2 | Clockwise arc feed | 
	G3 | Counter clockwise arc feed | 
	G4 | Dwell | 
	G10 L2 | Set offset parameters |
